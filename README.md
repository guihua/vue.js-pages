vue.js-pages
===

vue.js 多页面开发模板。

开始之前，请执行 node 运行环境的安装。

1. 新建项目
```bash
# 在 views 下新建项目目录
home
```

项目目录结构如下：
```bash
| - home
  | - index.html
  | - index.js
  | - company.vue
```


2. 项目运行
```bash
npm run dev
```

访问路径为：http://localhost:8080/home.html

3. 项目打包
```bash
npm run build
```

打包的资源文件在 dist 目录下。
